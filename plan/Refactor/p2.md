🎯 PROMPT CHO PHẦN 2: CHIA NHỎ VIDEO & GHÉP NỐI (CHUNK PIPELINE)
Vai trò: Ngươi là một Python Backend & FFmpeg Expert.
Nhiệm vụ: Trong file src/pipeline/orchestrator.py và renderer.py, hãy thực hiện chia nhỏ dự án video để render an toàn.
Yêu cầu chi tiết:

Viết logic duyệt qua mảng Scenes trong JSON gốc. Gom nhóm các Scene vào các file JSON nhỏ (chunk_1.json, chunk_2.json...).

Rule quan trọng: Thời lượng mỗi chunk khoảng 60-90 giây. Tuyệt đối KHÔNG cắt ngang một Scene đang diễn ra. Phải ngắt chunk ở điểm kết thúc của một Scene để không làm đứt file Audio.

Trong renderer.py, viết vòng lặp gọi lệnh Remotion CLI cho từng chunk (sử dụng subprocess). Cờ bắt buộc: --concurrency=2.

Viết hàm dùng lệnh FFmpeg concat demuxer (ffmpeg -f concat -safe 0 -i list.txt -c copy final.mp4) để nối siêu tốc các file chunk mp4 lại với nhau.

Viết Unit test đảm bảo tổng thời lượng các chunk khớp 100% với thời lượng video gốc.
